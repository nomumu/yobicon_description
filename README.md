### STLファイルについて

meshesフォルダ内のSTLファイルは誌面のように3Dプリンタで印刷することもできますが、噛み合わせの調整などの追加工が若干必要になります．
また3Dプリンタの個体差による誤差もあるため、完全に作例を再現出来ない可能性が高いです。
RViz以外で使用する場合は、このような点に注意して自己責任で利用して下さい。
